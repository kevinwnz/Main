Image Swap - On Mouseover + code using css to do rollover effect

The first example use attributes 'Onmouseover' and 'Onmouseout' to specify the images this uses code from the Document Objet Modal 'this.src='



Below is the source of the above rollover


<img src="img/gallery6.jpg" alt="Mouse Over Test" class="imgsize" title="Mouse Over Test"
onmouseover="this.src='img/gallery3.jpg'" onmouseout="this.src='img/gallery6.jpg'">

Note:   the code in red is the dhtml code used to fetch the mouseover


From: GeeksForGeeks

How this works it uses css code in the head section of this page a class called 'sudo' defines the main image,
(for this the two images must be same size) for the mouseover effect to work there is a sub-class called 'sudo:Hover' to replace the image on mouse over
