I have started a project to create an Animated Banner using the image editors I have, this how I did it and what I learned

What I have done so far...

  1. The first thing I did from GIMP Was to create a base banner, from which I can make my frames from. I am having great trouble trying to make my frames look as so it is typing out the words one letter at a time, I have even tried other image editors to achive this but have not had any luck so far.
  2. After playing around with various image editors I have found an editor that can do what I want and I am starting to build up the frames for the banner. The next step is to make the '.gif' file
  3. I have played with serveral ways of making the final animated '.gif' file. I now sorting out issues regarding getting the animation to look right, below this part of the page I am going to display the animations created with various programs and will comment on how easy it has been to get the animation working. (I may even change the layout of the page so I can have the animations on the right hand side of the screen. I am still making my mind up on how the final layout may appear)
  4. Now that I have got the this page sorted, The program I settled on to create the frames was 'Photofilter' which is from a french author. because this is a mainly txt based banner I had to find a way of writing the text I wanted on the banner and this editor suited my needs. the next thing was to assemble the 'frames' of which there are 39 and get the timings right, this why I tried 3 different methods of doing this. As you can see in the animation I did not get the text aligned properly so in the final animation in places it looks a bit strange.
  5. On the right hand side is the 3 examples of the banner done with the 3 different programs. I was going to show the frames used in the animated banner, but because of the size it was in practicle to display the frames
