Animated Banner and how I made it

I wanted to challenge myself so I created a project to create a animated Banner and document the processes I went through to create it. I also mentioned the programs I used.
I used a program called 'Photofilter' to create the frames for the animation. In order to create the animation you first need to create what is known as 'frames' these create the moving text / image that appear in the final animation.
After you have all the 'frames' you need to import them into the program that is going to make the final animations and depending on the program you choose you may have set the timing on each 'frame' or set a generic value which will apply to all 'frames', you then have save the file in a format that will run 'frames' as a single file there for getting the final animation.
below is one of the animation, in total I used 3 programs to create the animation click here to go the page on my wesite

Some things I have found with VS Code

Vs Code has a command pallet called Emmet which makes it easy to add command such as Paragraph or [p].
If you type a letter or letters related to a command in htm.
you will be prompted to complete the tag press the enter to complete.
When entering an html attribute if ctrl + space will bring up a list of classes, Id's, pages (.html), images
you can also do something like This 'ul>li*5' what it will produce is bulleted list with 5 list items
New! One thing I have found out with working with 'Iframes' is that to center the object you set the margins to auto and use the 'display' property and set it to 'Block' mode
(this is called Emmet and it make entering code easier)
I am still discovering other things about VS Code.

Hover-over image (onmouseover="this.src='image name'")

I had always wanted to revive the hover over image used on our family business website, I intially could not remember how I did it, but after doing a bit of searching I found what I wanted.
using the image tag and on mouseover and onmouseout with 'this.src='filename'' with in the 'a' tag
The other way I have demoed is using css code do the rollover image. I have had to put the css code in the head tag of the page This has caused me to create a seperate page to house the code. to see the page Click Here

 

Below is the source of the above rollover


<img src="img/gallery6.jpg" alt="Mouse Over Test" class="imgsize" title="Mouse Over Test"
onmouseover="this.src='img/gallery3.jpg'" onmouseout="this.src='img/gallery6.jpg'">

Note:   the code in red is the dhtml code used to fetch the mouseover
